# reactJS-fb-app

A Facebook-inspired social media UI built with React.js.

## 🔗 Live Demo

👉 [https://fb-frontend-ten.vercel.app/](https://fb-frontend-ten.vercel.app/)

## Features

- **News Feed** — Create posts with text and images, like, comment, and share
- **Profile Page** — Cover photo, stats, Posts / About / Friends / Photos tabs
- **Messenger** — Floating chat window, click any contact to open a conversation
- **Notifications** — Dropdown with recent activity
- **Friend Requests** — Confirm or delete requests from the topbar
- **People You May Know** — Add / unfriend from the rightbar
- **Watch** — Video grid with a click-to-play player
- **Groups** — Join / Leave toggle
- **Marketplace** — Listings with Contact Seller button
- **Saved** — Unsave items from your saved list
- **Events** — RSVP with Interested / Going toggle
- **Responsive sidebar** — See More / See Less navigation

## Tech Stack

- React.js 18
- React Router DOM v7
- Material UI (MUI) v5
- CSS (no external UI framework for layout)
- Deployed on Vercel

## Getting Started

```bash
npm install --legacy-peer-deps
npm start
```

## Build

```bash
npm run build
```
